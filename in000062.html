Skyloop

Ahana Gupta mail

Certification Date

Country

This is an iteration of the Skyloops project with added gamification and a scoreboard. We continued to use a checkpoint but made a square instead of a hexagon. We are also using an RGB LED strip to change colors based on whether the drone has passed through. The scoreboard will track how many times the player has flown the drone through the loop. The hoop will start off green, then begin blinking red to signal that the player must start. Afterward, the hoop will return to green.
